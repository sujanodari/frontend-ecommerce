# Ecommerce-Frontend

> ReactJS, TypeScript, GraphQL

## Installation Guide
```
# clone repository
$ git clone 

# install dependencies
$ npm i

# copy and modify the .env.example to .env

$ cd ecommerce

# start 
$ npm start

```

## Prettier

```
# Prettier Check 
$ npm run prettier:check

# Prettier Fix 
$ npm run prettier:fix 
```

## Create Build

```
$npm run build
```
